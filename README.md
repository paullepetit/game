# Game in Progress

## EaselJS - Tuto 2 : Sprite(no bull)sheets

### Présentation

Ce projet s'attarde sur l'affichage et la manipulation de sprites avec la bibliothèque EaselJS. Il présente les fonctionnalités suivantes :
* manipuler les Sprites et les Spritesheets,
* adapter les animations d’un Sprite aux événements claviers.
